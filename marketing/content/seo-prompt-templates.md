# SEO Content Plan — Prompt Templates by City

Purpose: publish optimized prompt guides that rank for long-tail "room for rent <city> under <budget>" queries while educating users how to phrase natural-language searches.

Each article should follow the same structure:
1. H1: `RentFinder Prompt Playbook: <City> Rentals`
2. Intro paragraph (80-120 words) describing the rental context + why natural-language saves time.
3. Prompt table (Budget / Location / Extras / Prompt example).
4. Screenshot or code block showing AI summary output.
5. CTA block linking to `/rent/<city>` and the main search bar.
6. FAQ (3 questions pulled from community comments).
7. Internal links: relevant `/rent/<other-city>` pages + blog posts.
